# Mobile Lead Form CTA Overlap Plan

## Scope

Improve mobile lead capture usability so sticky CTAs do not compete with or cover embedded forms.

## Planned Changes

- Hide the global mobile sticky CTA when any lead form is visible in the viewport.
- Keep the sticky CTA available elsewhere on conversion pages.
- Add breathing room inside sticky-submit modal forms so the sticky submit button does not crowd the last fields.
- Re-test mobile widths around 375px, 390px, and 768px.

## Out of Scope

- No CTA label changes.
- No new form fields.
- No backend changes.
- No deployment.
- No GitHub push.
