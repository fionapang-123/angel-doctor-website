# Restrict Palette Cleanup Plan

## Task

Clean up the site palette so it stays close to the four selected Option C guide colors.

## Scope

- Make pink backgrounds lighter and more restrained.
- Remove yellow/amber UI backgrounds.
- Map extra accent tokens back to the four guide colors:
  - `#d81b92`
  - `#2664d9`
  - `#eef6ff`
  - `#101828`
- Keep white and neutral border/text colors where needed for readability.
- Do not change copy, CTA labels, schema, or routing.

## Routes affected

- Global styling and treatment pages.

## Build plan

- Run `npm run build`.
