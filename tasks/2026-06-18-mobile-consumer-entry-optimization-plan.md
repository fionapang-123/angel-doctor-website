# Mobile Consumer Entry Optimization Plan

## Scope

Improve the mobile-first consumer entry experience for the primary lead capture paths.

## Planned Changes

- Delay the homepage mobile sticky CTA until after the hero section so it does not cover trust proof.
- Move the escort booking form before large team imagery on mobile while keeping the desktop layout unchanged.
- Remove sticky positioning from modal form submit buttons so they do not cover text fields.
- Re-test lint, build, and key mobile states.

## Out of Scope

- No CTA label changes.
- No backend changes.
- No new routes.
- No deployment.
- No GitHub push.
