# Homepage Hero Video Background Plan

## Task
Use the provided video as a lightweight homepage hero background while preserving GEO / SEO content as visible HTML text.

## Scope
- Review the current homepage hero implementation.
- Extract a short, web-friendly background clip from the provided video.
- Create a static poster image fallback.
- Update the homepage hero to use the video as visual background only.
- Keep H1, intro copy, CTA, trust cues, and compliance language in real HTML.
- Preserve approved CTA labels: Get Your Care Plan and Book Local Medical Escort.

## Compliance Notes
- Do not imply Angel Doctor provides diagnosis or treatment.
- Do not add fake hospitals, doctors, prices, testimonials, ratings, guaranteed appointments, or guaranteed outcomes.
- Keep medical decisions attributed to licensed healthcare providers.

## Validation
- Run `npm run build`.
- Check the homepage at mobile and desktop widths if a local preview is available.
