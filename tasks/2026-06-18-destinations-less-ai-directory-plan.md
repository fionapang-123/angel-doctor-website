# Destinations less-AI directory plan

## Goal

Make the `/destinations` city section feel less AI-generated and more like a curated medical travel directory.

## Scope

- Replace the oversized repeated city card grid with a tighter destination directory layout.
- Remove visible hospital count badges from the city overview section.
- Keep city names, treatment-fit terms, descriptions, and links crawlable for GEO.
- Preserve the China map module and conversion flow.

## Routes affected

- `/destinations`

## Out of scope

- Editing individual destination city pages.
- Changing hospital data.
- Adding new cities or routes.
- Creating new CTA labels.

## Verification

- Run `npm run build`.
