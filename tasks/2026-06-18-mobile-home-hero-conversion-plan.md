# Mobile Home Hero Conversion Plan

## Scope

Optimize the homepage hero for mobile consumers while keeping the desktop hero unchanged.

## Planned Changes

- Add a mobile-only hero content block with a short benefit line and primary CTA.
- Compress mobile trust proof into a compact strip instead of stacked cards.
- Keep the existing desktop hero title, video, and stat cards unchanged.
- Re-test mobile hero and desktop build.

## Out of Scope

- No desktop hero redesign.
- No CTA label changes.
- No backend changes.
- No deployment.
- No GitHub push.
